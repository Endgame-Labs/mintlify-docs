Individual users have the ability to upload documents to enhance the context provided to Endgame for singular account chat responses. Some types of documents that could be useful are account plans, meetings notes, or partnership decks. Anything that isn't already captured in the mediums Endgame already ingests can be added here.

### Adding a document from your computer

In a account detail page, open the Files tab toward the bottom of the page. Click Add Files and select Upload from device.

Go through the flow to select up to 10 files at a time from your device. You can edit the titles and descriptions for the files before clicking to Upload Files.

### Adding a document from Google Drive

In a account detail page, open the Files tab toward the bottom of the page. Click Add Files and select Link from Google Drive.

<Frame caption="File Upload">
  <img
    src="/mintlify/images/files-upload-8-30-25.png"
    alt="files upload"
    className="rounded-lg"
  />
</Frame>

Go through the Google authentication process. Select all permissions requested (we only requst [drive.file](https://www.googleapis.com/auth/drive.file) and [drive.readonly](https://www.googleapis.com/auth/drive.readonly) permissions) for the best Endgame experience. Once athenticated, select up to 10 files at a time to sync with Endgame. You can edit the description for the files before clicking to Select Files.
